Time API
========

A simple REST API that gives you the current time and date in GMT.

Example:

`curl http://peaceful-dawn-5311.herokuapp.com/time`

```
{
  "timezone": "GMT",
  "date": "Mon Aug 04 2014",
  "time": "9:23:42",
  "hour": 9,
  "minute": 23,
  "second": 42,
  "year": 2014,
  "month": 7,
  "day": 4
}
```
